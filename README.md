# Instagram RicardoImp Dashboard

Dashboard **mobile first** para leitura estratégica do perfil **@ricardoimp**, com foco em quatro camadas: visão executiva, análise profunda, inteligência de audiência e motor de decisão.

## O que este painel entrega

O projeto transforma os dados disponíveis em uma leitura prática de crescimento e posicionamento. A interface foi construída com direção visual **editorial executivo cinematográfico**, priorizando autoridade, clareza e acionabilidade.

## Fontes usadas nesta versão

| Fonte | Uso no painel | Status |
| --- | --- | --- |
| Instagram conectado | Perfil, posts recentes e leitura estratégica de conteúdo | Ativo |
| Meta Ads conectado | Estado das contas e diagnóstico dos últimos 30 dias | Ativo, porém sem insights retornados no período |

## Estrutura do dashboard

| Camada | Entrega |
| --- | --- |
| Resumo executivo | Síntese do momento atual do perfil e do estado da mídia paga |
| O que está acontecendo | Performance por formato, ranking inteligente e diagnóstico de posicionamento |
| O que isso significa | Leitura de audiência e padrões ocultos |
| O que fazer agora | Erros, oportunidades, plano semanal, testes A/B e alertas |

## Atualização contínua

A intenção do projeto é manter o painel atualizado **todo domingo**, sempre exibindo a data da última atualização no topo da interface.

## Observação importante

Nesta versão, alguns blocos permanecem marcados como **N/A** ou **dados indisponíveis** porque os conectores atuais não retornaram determinadas métricas do período consultado. Essas lacunas foram mantidas visíveis de propósito para evitar qualquer número inventado.
